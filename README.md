VIRINCHI 2K26 – Event Website

VIRINCHI 2K26 is a responsive university fest website built to showcase technical and cultural events across multiple departments. The platform features dynamic event rendering, interactive flip cards, expandable event themes, and a modern UI design optimized for all devices.

Features

Department-based event filtering (CSE, ECE, EEE, ME)

Interactive 3D flip event cards

Expandable themes section for Poster Presentation

Icon-based event details (Date, Venue, Participation)

Responsive design (Desktop, Tablet, Mobile)

Clean gradient-based UI with watermark branding

Tech Stack

HTML5

CSS3 (Flexbox, Grid, Media Queries)

JavaScript (Vanilla JS)

Font Awesome Icons

Purpose

Designed to provide an intuitive and professional digital platform for managing and displaying university fest events.
